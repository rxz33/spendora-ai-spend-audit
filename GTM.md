# Go-to-Market Strategy for Spendora

## Target User Profile

**Exact User:** Engineering Manager or Staff Engineer at Series A/B SaaS startups (funded, <50 engineers)

**Why this user:**
- Making procurement decisions (budget authority)
- Directly responsible for AI tool subscriptions
- Feels cost pressure from leadership
- Has team context (multiple developers, multiple tools)
- Recent experience with tool sprawl

**Company Stage:** Series A/B ($5M-$30M funding)
- Larger than pre-seed (no budget to allocate)
- Not yet enterprise (don't care about $500/month)
- Actively shipping products (high tool usage)
- Cash-conscious but not bootstrapped

## Go-to-Market Plan: $0 Budget, 100 Users in 30 Days

### Week 1: Launch + Seed Community (Target: 20 users)
**Action:** Post on Reddit + Indie Hackers
1. **r/startups** — "I built a free audit tool for AI tool spend. Here's what I learned from 3 founder interviews..."
   - Be honest about findings
   - Show 1-2 real quotes from interviews
   - Link to tool
   - Expected: 5-8 upvotes, 50-80 clicks, 8-12 users

2. **Indie Hackers** — "Free AI spend audit tool (no sign-up required, shareable results)"
   - Show clean screenshot
   - Explain it took 3 days to build
   - Link to deployed app
   - Expected: 30-50 clicks, 8-12 users

3. **r/SideProject** — Post your build
   - Mention you're an intern at Credex
   - Show honest user interview insights
   - Expected: 20-30 clicks, 4-6 users

4. **Direct outreach** — DM 10 founders on X with personalized pitch
   - "I built this specifically because I talked to 3 founders who didn't realize..."
   - Expected: 2-3 responses, 1-2 signups

### Week 2-3: Amplification + Referral (Target: 50-60 cumulative users)
1. **Founder friend referrals** — Ask 3-5 founder friends to share
2. **Post in founder Slack groups** — If you have access, share in 2-3 groups
3. **HN (if ready)** — Submit to Hacker News if confident (500+ users possible if hits front page)
4. **Twitter thread** — Share a real audit example + metrics
5. **Follow-up posts** — Reddit weekly updates with metrics: "Built for free, got 50 users. Here's what I learned..."

### Week 4: Credex Connection (Target: 80-100+ users)
1. **Credex promotion** — Credex mentions tool in newsletter/social
2. **Partner channels** — Any existing Credex community/audience shares

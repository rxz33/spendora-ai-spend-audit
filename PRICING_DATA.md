# OpenAI API (GPT-4o mini)

Token Pricing:
- Input: $0.15 / 1M tokens
- Output: $0.60 / 1M tokens

Usage Estimates (per developer per month):
- Light coding (5 hours/week): ~10,000 tokens = ~$0.01/month
- Medium coding (20 hours/week): ~40,000 tokens = ~$0.03/month
- Heavy coding (40 hours/week): ~100,000 tokens = ~$0.07/month

**Cost Analysis:**
- ChatGPT Plus: $20/month (unlimited usage)
- API usage (medium): ~$0.03/month per developer
- Breakeven: ChatGPT Plus pays for itself at ~650 days of API access

**Recommendation Logic:**
- Teams with consistent, moderate usage → API is cheaper
- Example: 8-person team using ChatGPT Plus ($20/month) vs API ($2.40/month)
- Savings: $17.60/month or $211.20/year per subscription

Official Pricing:
https://openai.com/api/pricing

Verified:
2026-05-11

# Anthropic API (Claude 3.5 Sonnet)

Token Pricing:
- Input: $3 / 1M tokens
- Output: $15 / 1M tokens

Usage Estimates (per user per month):
- Light writing (3 hours/week): ~20,000 tokens = ~$0.12/month
- Medium writing (10 hours/week): ~60,000 tokens = ~$0.30/month
- Heavy writing (30 hours/week): ~150,000 tokens = ~$0.75/month

**Cost Analysis:**
- Claude Pro: $20/month (unlimited usage)
- API usage (medium): ~$0.30/month per user
- Breakeven: Claude Pro pays for itself at ~67 days of API access

**Recommendation Logic:**
- Teams with document-heavy workflows → API is cheaper
- Example: 5-person team using Claude Pro ($20/month × 5 = $100/month) vs API (~$1.50/month)
- Savings: $98.50/month or $1,182/year per subscription

Official Pricing:
https://www.anthropic.com/pricing#api

Verified:
2026-05-11
